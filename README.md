# Dotfiles

Repo contains the dotfiles for my linux/mac setup

## Requirements

Check for the following:

### Git

To ensure version control
For arch:
```
sudo pacman -S git
```
For debian:
```
sudo apt install git
```
### Stow

Allows us to have linked dotfiles in a single directory
For arch:
```
sudo pacman -S stow
```
For debian:
```
sudo apt install git
```
